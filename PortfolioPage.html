    Mukicik Music Store

    Mukicik is an online store website that sells musical instruments. In this project, I developed a dynamic website as an online shop platform for this music shop. This website has full features like an online shop website, with features including user registration, user login, users can change profile pictures, insert items if the user wants to sell their musical instruments, and update items to update data on musical instruments that want to sell. Because this is a dynamic website, this website displays a different view between the logged in user and the guest user. Logged in user will get full access to the website. Some minor details such as error handling in the form of input validation and error messages preventing input errors by the user, as well as all item data and user credentials stored in the database.

    Asp.net, Html, Css, C#, SQL
